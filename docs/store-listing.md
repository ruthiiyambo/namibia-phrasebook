## Full Description Draft

Namibia Phrasebook is a practical mobile phrasebook built for real travel moments.

Browse useful phrases in Afrikaans, Oshiwambo, and Nama-Damara for greetings, food,
directions, transport, health, numbers, and local etiquette. The app is designed to help
you find the right phrase quickly instead of scrolling through one long list.

### What you can do

- Browse phrases by situation and category
- Search by meaning, local phrase, pronunciation, or topic
- View pronunciation help and usage notes
- Use the launch build without creating an account
- Access the bundled phrasebook offline after installation

### Good fit for

- First-time visitors to Namibia
- Travelers moving between towns, lodges, shops, and transport
- Anyone who wants fast practical language help without setup friction
